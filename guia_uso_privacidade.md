# Guia de Uso e Privacidade - Meu Alley

> **Resumo**: O Meu Alley **não coleta nenhum dado pessoal**. Todos os dados são armazenados apenas no seu dispositivo, funcionando completamente offline. Não há coleta, rastreamento, cookies ou compartilhamento de dados com terceiros.

## Sobre o Aplicativo

**Meu Alley** é um aplicativo desenvolvido para ajudar você a gerenciar suas informações sobre os artistas em eventos. Com ele, você pode organizar seus artistas favoritos, controlar seus gastos, criar listas de tarefas e muito mais.

---

## Privacidade e Dados

### Coleta de Dados

Valorizamos sua privacidade e projetamos o Meu Alley para operar sem coleta de dados pessoais.

**Meu Alley** não coleta nenhum dado pessoal do usuário. O aplicativo funciona completamente offline e não se conecta a nenhum servidor externo.

### Dados Armazenados Localmente

Todos os dados que você cria no aplicativo são armazenados **exclusivamente no seu dispositivo** usando o sistema de armazenamento local do sistema operacional. Esses dados incluem:

- **Lista de artistas favoritos**: IDs dos artistas que você marcou como favoritos
- **Itens para autógrafo**: Lista de itens que você deseja autografar por cada artista
- **Gastos registrados**: Valores e descrições dos gastos que você adicionou
- **Orçamento definido**: Valor total do orçamento que você configurou
- **Dias selecionados**: Dias que você selecionou para cada artista
- **Lista de tarefas**: Tarefas que você criou na aba ToDo

### Segurança dos Dados

- Todos os dados são armazenados localmente no seu dispositivo
- Os dados não são transmitidos para nenhum servidor
- Os dados não são compartilhados com terceiros
- Os dados não são usados para publicidade ou análise

### Exclusão de Dados
Você pode excluir todos os dados do aplicativo a qualquer momento:
- **Android**: Vá em Configurações > Apps > Meu Alley > Armazenamento > Limpar Dados
- **iOS**: Desinstale e reinstale o aplicativo

**Nota**: A exclusão dos dados é permanente e não pode ser desfeita. Todos os seus favoritos, gastos, tarefas e configurações serão perdidos.

### Permissões do Aplicativo

O aplicativo não solicita nenhuma permissão sensível ou relacionada a dados pessoais.

- **Acesso à Rede Local** (apenas para desenvolvimento): Usado apenas durante o desenvolvimento do aplicativo para debugging. Não é usado em produção.

### Cookies, SDKs de Terceiros e Rastreamento

**Meu Alley** não utiliza:
- **Cookies**: O aplicativo não usa cookies de nenhum tipo
- **SDKs de terceiros**: Não há integração com SDKs de analytics, publicidade ou rastreamento
- **Bibliotecas de rastreamento**: Não há coleta invisível de dados ou rastreamento de comportamento do usuário
- **Serviços de analytics**: Não há integração com Google Analytics, Firebase Analytics ou serviços similares
- **Publicidade**: O aplicativo não exibe anúncios e não utiliza SDKs de publicidade

O aplicativo funciona completamente offline e não se conecta a nenhum servidor externo, garantindo que nenhum dado seja coletado ou rastreado.

### Informações dos Artistas

O aplicativo exibe informações públicas sobre artistas participantes de eventos (como nomes, biografias, fotos, números de mesa, dias disponíveis e links para redes sociais). Essas informações são:

- **Estritamente informativas**: Fornecidas apenas para ajudar você a planejar sua visita ao evento
- **Dados públicos**: Informações já disponíveis publicamente sobre os artistas e o evento
- **Sem vínculo pessoal**: Não há nenhuma relação ou vínculo pessoal entre essas informações e você como usuário
- **Pré-carregadas**: Essas informações vêm incluídas no aplicativo e não dependem de dados pessoais seus
- **Não coletadas do usuário**: O aplicativo não coleta, armazena ou transmite informações pessoais dos usuários relacionadas aos artistas

As informações dos artistas são meramente exibidas para sua conveniência e não contêm nem estão vinculadas a dados pessoais dos usuários do aplicativo.

### Tratamento de Dados Pessoais e LGPD

**Meu Alley não trata dados pessoais** conforme definido pela Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

#### Dados Não Identificáveis

Todos os dados armazenados localmente no aplicativo são **não identificáveis** e não permitem a identificação direta ou indireta de uma pessoa natural. Os dados armazenados incluem apenas:

- Preferências e configurações do aplicativo (IDs de artistas favoritos, valores de gastos, textos de tarefas)
- Informações que não contêm elementos que possam identificar o usuário (como nome, CPF, email, telefone, endereço, etc.)

Esses dados são armazenados exclusivamente no dispositivo do usuário e não são transmitidos ou compartilhados com terceiros.

#### Direitos da LGPD

Como o aplicativo **não trata dados pessoais**, os seguintes direitos previstos na LGPD **não se aplicam** ao uso deste aplicativo:

- Confirmação da existência de tratamento de dados pessoais
- Acesso aos dados pessoais
- Correção de dados pessoais incompletos, inexatos ou desatualizados
- Anonimização, bloqueio ou eliminação de dados pessoais desnecessários, excessivos ou tratados em desconformidade com a LGPD
- Portabilidade dos dados pessoais
- Eliminação dos dados pessoais tratados com consentimento
- Informação sobre compartilhamento de dados pessoais
- Informação sobre a possibilidade de não fornecer consentimento e sobre as consequências da negativa
- Revogação do consentimento

**Observação**: Embora esses direitos não se apliquem ao tratamento de dados pessoais (pois não há tratamento), você pode excluir todos os dados armazenados localmente no aplicativo a qualquer momento através das configurações do seu dispositivo, conforme descrito na seção "Exclusão de Dados".

## Dicas de Uso

### Planejamento da CCXP

1. **Antes do evento**:
   - Explore a lista de artistas e marque seus favoritos
   - Defina seu orçamento na aba Gastos
   - Crie tarefas na aba ToDo para organizar o que você quer fazer

2. **Durante o evento**:
   - Use o agrupamento por mesa para planejar sua rota
   - Registre seus gastos conforme você compra
   - Marque tarefas como concluídas conforme você as realiza
   - Adicione itens para autógrafo conforme necessário

3. **Filtros úteis**:
   - Use o filtro por tipo para focar em áreas específicas do evento
   - Combine busca e filtros para encontrar rapidamente o que precisa
